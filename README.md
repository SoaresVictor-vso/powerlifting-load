# Como executar

## Linux
* Abrir o terminal dentro da pasta;
* Executar o comando `chmod +x powerlifting-loader-v1-linux`, garantindo permissão de execução ao arquivo;
* Clique duplo sobre o arquivo powerlifting-loader-v1-linux

## Windows
* Clique duplo sobre o arquivo powerlifting-loader-v1-windows-64


# Como jogar

## Visão Geral

O powerlifting é um esporte de força no qual o alteta deve levantar o máximo de peso que for capaz para uma repetição nos exercícios agachamento, supino e levantamento terra. Para todos estes, usa-se uma barra de 20kg e um par de presilhas de 2,5kg cada.

O atleta realiza sua pedida escolhendo um peso maior ou igual a 25kg (barra + presilhas), sempre multiplo de 0,5kg. As anilhas devem ser dispostas sempre da mais pesada (ao centro), para a mais leve (nas bordas). Sempre deve ser usada a maior anilha possível, de forma que não exceda o peso.

As anilhas têm cores definidas:

* 25kg: vermelho
* 20kg: azul
* 15kg: amarelo
* 10kg: verde
* 5kg: livre
* 2,5kg: livre
* 1,25kg: livre
* 0,5kg: livre
* 0,25kg livre

## Instruções

Ao clicar em iniciar, um peso aleatório, dentro das regras será sorteado. Há um botão com cada cor de anilha e seu respectivo peso. Ao clicar nele, um par de anilhas do peso selecionado é adicionado à barra de forma irreversível (ao clicar em 25kg, será adicionado 50kg, sendo uma anilha desse peso de cada lado).

Após adicionar as anilhas necessárias, deve-se clicar no botão "Barra Pronta". Após isso, aparecerá o peso total, que seria quanto peso foi adicionado à barra, que deve coincidir com o objetivo.

Caso tenha acertado, aparecerão 3 círculos brancos, caso contrário, 3 vermelhos e um botão de "mostrar correta", que apresenta a forma correta de como a barra deveria ter sido montada.

Após a finalização, aparece o botão "limpar", que volta ao estado inicial e permite que se inicie uma nova partida.

Uma observação pertinente: Se o objetivo for 105kg e se adicionar 2 anilhas de 20kg de cada lado (4x20kg = 80kg) + 25kg (barra + presilhas), o peso total será correto, mas o jogador receberá 3 luzes vermelhas, pois não usou a maior anilha possível. Nesse caso, a forma correta seria uma anilha de 25kg e uma de 15kg de cada lado, resultando no mesmo peso total, porém agora, seguindo as regras.
